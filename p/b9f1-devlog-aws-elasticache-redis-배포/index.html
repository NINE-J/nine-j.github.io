AWS ElastiCache Redis л°°нҸ¬

Spring Boot л¶„мӮ° нҷҳкІҪ Redis AWS мҡҙмҳҒ кө¬м„ұ

2025л…„ 10мӣ” 16мқј
3 minute read

рҹ“Ңк°ңмҡ”

нҳ„мһ¬ н”„лЎңм қнҠём—җм„ңлҠ” PostgreSQL(RDS), Kafka(Confluent Cloud)мҷҖ н•Ёк»ҳ RedisлҘј м„ём…ҳ/мәҗмӢңмҡ©мңјлЎң мҡҙмҳҒн•ҙм•ј н•ҳлҠ” мғҒнҷ©мқҙлӢӨ.

RedisлҘј лЎңм»¬м—җм„ң docker-composeлЎң н…ҢмҠӨнҠёлҠ” н–Ҳм§Җл§Ң мҡҙмҳҒ нҷҳкІҪм—җм„ңмқҳ м—°кІ° кө¬мЎ°мҷҖ м„Өм • мӣҗлҰ¬лҘј лӘ…нҷ•нһҲ мқҙн•ҙн•ҳкі мһҗ н•ңлӢӨ.

нҳ„мһ¬ кі„нҡҚмқҖ ECS Fargate кё°л°ҳ л¶„мӮ° нҷҳкІҪмқҙл©° RedisлҘј м–ҙл–»кІҢ л°°нҸ¬н•ҳкі  м—°кІ°н•ҳлҠ”м§Җ к°„лӢЁнһҲ м •лҰ¬н•ҙліёлӢӨ.

рҹ“ҢлӮҙмҡ©

RedisлҠ” кё°ліё кө¬м„ұл§ҢмңјлЎң лҸҷмһ‘н•ңлӢӨ

RedisлҠ” HTTP кё°л°ҳмқҙ м•„лӢҢ TCP кё°л°ҳмқҳ Key-Value м„ңлІ„лӢӨ. HTTPмІҳлҹј кІҪлЎң к°ңл…җмқҙ м—Ҷкі  лӢЁмҲңнһҲ м •мқҳлҗң нҸ¬нҠё(кё°ліё 6379)лЎң м—ҙл Ө мһҲлҠ” мҶҢмј“мқ„ нҶөн•ҙ лӘ…л №мқ„ мЈјкі  л°ӣлҠ”лӢӨ.

мҰү, RedisлҠ” ліөмһЎн•ң мқёмҰқмқҙлӮҳ н•ёл“ңм…°мқҙнҒ¬ м—ҶмқҙлҸ„ нҒҙлқјмқҙм–ёнҠёк°Җ IPмҷҖ нҸ¬нҠёлҘј м•Ңкі  мһҲмңјл©ҙ л°”лЎң м ‘к·ј к°ҖлҠҘн•ң кө¬мЎ°лӢӨ.

  • RedisлҠ” host:port кё°л°ҳмқҳ TCP н”„лЎңнҶ мҪңл§Ң мһҲмңјл©ҙ лҸҷмһ‘н•ңлӢӨ.
  • лі„лҸ„мқҳ м„ңл№„мҠӨ л Ҳмқҙм–ҙк°Җ м—ҶмңјлҜҖлЎң м„Өм •мқҳ н•өмӢ¬мқҖ м–ҙл””лЎң л¶ҷмқ„ кІғмқёк°Җ.

Spring Bootм—җм„ң Redis м—°кІ° кө¬мЎ°

Spring BootлҠ” spring-boot-starter-data-redis мқҳмЎҙм„ұл§Ң м¶”к°Җн•ҳл©ҙ к°„лӢЁн•ҳкІҢ LettuceConnectionFactory, RedisTemplate, CacheManager л“ұмқ„ л“ұлЎқн•ңлӢӨ.

лі„лҸ„мқҳ Beanмқ„ л“ұлЎқн•ҳм§Җ м•Ҡм•„лҸ„ Spring BootлҠ” мқҙлҘј к°җм§Җн•ҙ мһҗлҸҷ м—°кІ°н•ңлӢӨ. м„Өм •мқ„ мғқлһөн•ҳл©ҙ кё°ліёк°’(localhost:6379)мңјлЎң мӢңлҸ„н•ңлӢӨ.

Elastic Redis л°°нҸ¬ к°ңмҡ”

AWSм—җм„ңлҠ” ElastiCache Redis OSSлҘј мӮ¬мҡ©н•ҳл©ҙ м„ңлІ„ м„Өм№ҳ м—Ҷмқҙ кҙҖлҰ¬нҳ• Redis нҒҙлҹ¬мҠӨн„°лҘј мӮ¬мҡ©н•  мҲҳ мһҲлӢӨ.

кө¬м„ұ мҡ”мҶҢм„ӨлӘ…
Primary Nodeм“°кё° лӢҙлӢ№ Redis л…ёл“ң
Replica NodeмқҪкё° м „мҡ© ліөм ңліё (м„ нғқмӮ¬н•ӯ)
Clusterм—¬лҹ¬ мғӨл“ңлЎң лҚ°мқҙн„°лҘј л¶„мӮ° м ҖмһҘ
Endpointм—°кІ°мҡ© мЈјмҶҢ (.cache.amazonaws.com)

ECS Fargate м—°кІ° кө¬мЎ°

  • ліҙм•Ҳ к·ёлЈ№ к·ңм№ҷ
    • Inbound: 6379 н—Ҳмҡ© (ECS -> Redis)
    • Outbound: ECSм—җм„ң Redis SG(Security Group)лЎң м ‘к·ј н—Ҳмҡ©
  • VPC: к°ҷмқҖ VPC лҳҗлҠ” н”јм–ҙл§Ғлҗң л„ӨнҠёмӣҢнҒ¬м—җ мЎҙмһ¬н•ҙм•ј н•Ё

лЎңм»¬ DockerмҷҖ мҡҙмҳҒ Redisмқҳ кҙҖкі„

кө¬л¶„лЎңм»¬ нҷҳкІҪмҡҙмҳҒ нҷҳкІҪ
нҳёмҠӨнҠёlocalhost:6379Elasticache Endpoint
лӘ©м Ғк°ңл°ң л°Ҹ лӢЁмң„ н…ҢмҠӨнҠёмӢӨм ң м„ңл№„мҠӨ мҡҙмҳҒ
кіөмң  м—¬л¶Җмҷ„м „нһҲ л¶„лҰ¬мҷ„м „нһҲ л¶„лҰ¬
м°Ёмқҙм җлӢЁмқј мқёмҠӨн„ҙмҠӨл©ҖнӢ° AZ, нҒҙлҹ¬мҠӨн„°л§Ғ, лӘЁлӢҲн„°л§Ғ м ңкіө

мҰү, мҪ”л“ңмғҒмңјлЎңлҠ” лҸҷмқјн•ң RedisTemplateмқ„ мӮ¬мҡ©н•ҳм§Җл§Ң нҷҳкІҪ ліҖмҲҳ(SPRING_DATA_REDIS_HOST, SPRING_DATA_REDIS_PORT)л§Ң лӢ¬лқјм§„лӢӨ.

нҳёмҠӨнҠёмҷҖ нҸ¬нҠёлҘј л§һм¶”м–ҙ TCP л ҲлІЁм—җм„ң м—°кІ°л§Ң лҗҳл©ҙ RedisлҠ” лі„лҸ„ ліөмһЎн•ң м„Өм • м—Ҷмқҙ л°”лЎң лҸҷмһ‘н•ңлӢӨ.

мҡҙмҳҒ мӢң кі л ӨмӮ¬н•ӯ

мҡҙмҳҒ нҷҳкІҪм—җм„ңлҠ” лӢӨмқҢ н•ӯлӘ©л“Өмқ„ нҶөн•ҙ ліҙм•Ҳкіј м•Ҳм •м„ұмқ„ к°•нҷ”н•  мҲҳ мһҲлӢӨ.

кө¬л¶„м„ӨлӘ…
ліҙм•Ҳ к·ёлЈ№ м ңн•ңRedis нҸ¬нҠё(6379)лҠ” мҳӨм§Ғ ECS м„ңл№„мҠӨмқҳ ліҙм•Ҳ к·ёлЈ№м—җм„ңл§Ң м ‘к·ј к°ҖлҠҘн•ҳлҸ„лЎқ м„Өм •н•ңлӢӨ.
VPC лӮҙл¶Җ нҶөмӢ нҚјлё”лҰӯ м•Ўм„ёмҠӨлҘј л№„нҷңм„ұнҷ”н•ҳкі , VPC лӮҙл¶Җ нҠёлһҳн”ҪмңјлЎңл§Ң м ‘к·јмқ„ м ңн•ңн•ңлӢӨ.
м•”нҳёнҷ” мҳөм…ҳElasticache мғқм„ұ мӢң вҖҳм „мҶЎ мӨ‘ м•”нҳёнҷ”(Encryption in-transit)вҖҷ л°Ҹ вҖҳм ҖмһҘ мӢң м•”нҳёнҷ”(Encryption at-rest)вҖҷлҘј нҷңм„ұнҷ”н•ңлӢӨ.
Redis AUTH (м„ нғқ)OSS кё°ліёмқҖ мқёмҰқ лҜём§Җмӣҗмқҙм§Җл§Ң, н•„мҡ” мӢң Redis Enterprise лҳҗлҠ” н”„лЎқмӢң кі„мёөмқ„ нҶөн•ҙ л№„л°ҖлІҲнҳё мқёмҰқмқ„ кө¬м„ұн•  мҲҳ мһҲлӢӨ.
м„ём…ҳ кіөмң  м„Өм •Spring Boot м„ём…ҳ мҠӨнҶ м–ҙлҘј RedisлЎң ліҖкІҪ (spring.session.store-type=redis).
TTL кҙҖлҰ¬мәҗмӢң л§ҢлЈҢмӢңк°„мқ„ л°ҳл“ңмӢң м„Өм • (Duration.ofMinutes(...) л“ұ).
лӘЁлӢҲн„°л§ҒCloudWatch, Redis SlowLog, CloudWatch Metrics л“ұмқ„ нҷңмҡ©н•ҙ м§Җн‘ңмҷҖ м„ұлҠҘмқ„ м¶”м Ғн•ңлӢӨ.

рҹҺҜкІ°лЎ 

RedisлҠ” кё°ліём ҒмңјлЎң нҸ¬нҠёл§Ң л§һмңјл©ҙ лҸҷмһ‘н•ҳм§Җл§Ң,
мҡҙмҳҒ нҷҳкІҪм—җм„ңлҠ” ліҙм•Ҳ к·ёлЈ№, VPC, м•”нҳёнҷ” мҳөм…ҳмқ„ нҶөн•ҙ мөңмҶҢн•ңмқҳ л°©м–ҙм„ мқ„ л§Ңл“Ө мҲҳ мһҲлӢӨ.

Spring Bootм—җм„ңлҠ” hostмҷҖ portл§Ң м§Җм •н•ҳл©ҙ мһҗлҸҷмңјлЎң м—°кІ°лҗҳкі  Elasticache RedisлҠ” мқҙлҘј VPC лӮҙл¶Җм—җм„ң м•Ҳм „н•ҳкІҢ кҙҖлҰ¬н•ҙмӨҖлӢӨ.
